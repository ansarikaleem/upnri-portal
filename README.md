# UPNRI Forum Kuwait - Web Portal

A comprehensive web portal for UPNRI Forum Kuwait community with member directory, business listings, news/events, and admin panel.

## Features

- **Member Management**: Registration with Civil ID validation, profile management
- **Business Directory**: Member business listings with approval system
- **News & Events**: Dynamic content management with privacy controls
- **Gallery**: Event photos and media management
- **Admin Panel**: Complete website management system
- **Responsive Design**: Mobile-first approach with Tailwind CSS

## Tech Stack

### Frontend
- Next.js 13
- React 18
- Tailwind CSS
- Axios for API calls

### Backend
- Node.js with Express
- MySQL with Sequelize ORM
- JWT authentication
- Multer for file uploads
